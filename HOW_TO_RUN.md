# How to run the app

1. **Start the server** (in Terminal or Cursor terminal):
   ```bash
   cd "/Users/psonkusare/Documents/Beauty-playground "
   python3 -m http.server 8080
   ```

2. **Open in a real browser** (not Cursor’s built-in preview):
   - Open **Chrome**, **Safari**, or **Firefox**
   - Go to: **http://localhost:8080**

The in-editor browser often can’t connect to localhost. Use Chrome/Safari/Firefox to avoid “Connection refused”.
